### Other stuff
- It needs to be on both server and client
- Obviously, keybind for opening gui can be changed. There isn't a default so you have to set it yourself
- You can either drag an item into the slot from NEI or you can type the name of the part you wanna filter by
- You can get a report of your full network by leaving both options blank, you'll need to have report files on so you can go through it at your own pace
